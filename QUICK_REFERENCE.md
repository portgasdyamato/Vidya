# 🚀 Vidya AI - Quick Reference Card

## Start Server
```bash
npm run dev
```
**URL:** http://localhost:5000

---

## Upload Workflow

### 📄 Document Upload
1. Click "Upload Documents"
2. Select PDF/DOCX (max 10MB)
3. Choose options → Process

### 🎥 Video Upload  
1. Click "Video URL" tab
2. Paste YouTube/Vimeo link
3. Choose options → Process

## Troubleshooting

### Quick Fixes
```bash
# Check system health
node diagnostic.mjs

# Reset database
npm run db:push

# Restart server
npm run dev
```

### Common Issues
- **Upload fails** → Refresh page, try again
- **AI not responding** → Check GEMINI_API_KEY
- **Processing stuck** → Refresh page
